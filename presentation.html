🐄 Cattle Seller

Animal Marketplace Platform

A full-stack web application for buying/selling livestock & pets with AI price prediction, chat, reviews, and advanced search/filtering.

🎯 Project Overview

Cattle Seller is a comprehensive online marketplace for animals (cows, goats, camels, pets etc.). Users can post ads with multiple images, filter/search by category/city/price, chat with sellers, leave reviews, favorite listings, and get AI-powered price predictions based on weight/breed/color.

🛠️ Tech Stack

FastAPI SQLAlchemy (SQLite) React 18 Vite Tailwind CSS JWT Auth WebSocket Chat Scikit-learn ML Pydantic

✨ Key Features

  • 📱 Responsive Home Feed - Category carousel, advanced filters (type/city/price), search, sorting
  • 📤 Post Ad - Multi-image upload, rich form, AI price estimator
  • 💬 Real-time Chat - WebSocket messaging, unread notifications
  • ⭐ Reviews & Ratings - Seller ratings visible on listings
  • ❤️ Favorites - Save listings to personal collection
  • 🔮 AI Price Prediction - ML model predicts fair price
  • 🔐 Complete Auth - Signup/login/reset, profile image upload
  • 📊 Dashboard - My listings, analytics

🏗️ Architecture

Frontend (React)

  • ✅ App.jsx (Router)
  • ✅ Pages: Home, PostAd, Chat, Profile, Dashboard
  • ✅ Components: Navbar, Filters, AnimalCard
  • ✅ AuthContext

Backend (FastAPI)

  • ✅ /animals/ (CRUD)
  • ✅ /auth (JWT)
  • ✅ /ws/{user_id} (WebSocket)
  • ✅ /predict-price/ (ML)

🚀 API Endpoints

Auth

POST /signup
POST /login  
POST /forgot-password
POST /reset-password

Animals

POST /animals/
GET /animals/?type=cow&city=Lahore
GET /animals/{id}
DELETE /animals/{id}

Chat

WS /ws/{user_id}
POST /messages/
GET /messages/{user_id}

ML

POST /predict-price/
(weight, age, breed, color)
